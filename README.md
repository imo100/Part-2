Project Overview

The Charles Greenway website project involves the development of an effective, easy to use, gardening and landscaping website for the Charles Greenway business. The purpose of the Charles Greenway website will be to provide customers with an easy means to obtain information about gardening services, the prices for those services, the details of the business and contact methods.

The website aims to create and improve the Charles Greenway online presence and to create a simple method for customers to get in contact with Charles Greenway about services offered by the business. The Charles Greenway website will assist with customer enquiries via an online enquiry form, with clear contact details for the business.

The process involved in the project was: planning, research, creation of a sitemap, creation of HTML structures, stylisation of the website, and testing. The goal of the project is to build a visually appealing, responsive website, reflecting the nature of the business, that is also environmentally friendly.

Typography
Playfair Display will be used as the display and hero font for page titles and large headings. Font sizes will range between 36px and 52px to create a professional visual appearance.
Lato (Regular and Bold) will be used as the primary body font. Font sizes will range between 14px and 24px and will be used for body text, navigation menus, captions, and buttons because the font is clean and easy to read.
Lato Bold will also be used for subheadings and section titles. Font sizes will range between 18px and 28px and will be applied to H2 and H3 headings, service cards, and important content sections.
